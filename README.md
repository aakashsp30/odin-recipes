# Odin Recipes

A simple HTML recipe website project that demonstrates basic HTML structure, links, lists, and images.

## Description

This project is a basic recipe website built with pure HTML as part of The Odin Project curriculum. It consists of a main index page that links to individual recipe pages, each containing a recipe with an image, description, ingredients list, and preparation steps.

## Skills Demonstrated

- HTML boilerplate structure
- Headings (`<h1>`, `<h2>`)
- Paragraphs (`<p>`)
- Links (`<a>`) with relative paths
- Images (`<img>`)
- Ordered lists (`<ol>`)
- Unordered lists (`<ul>`)
- Proper file and directory organization
